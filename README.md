## ⚡ Quick Start

### Production Mode
```bash
export DEPLOY_ENV=production
./scripts/deploy.sh
```
## Development Setup
```bash
npm install
npm run dev
```

## Experimental (AI-Enhanced) Mode
```bash
# Install AI dependencies
pip install tensorflow keras
# Initialize AI models
./scripts/init-ai-models.sh
# Start with AI-enhanced mode
npm run start:ai
```
